## Introduction
백준 온라인 저지(BOJ)의 **[요세푸스 문제 0](https://www.acmicpc.net/problem/11866)** 문제 풀이입니다.

## Problem Description
> [문제 링크](https://www.acmicpc.net/problem/11866)

<p>요세푸스 문제는 다음과 같다.</p>

<p>1번부터 N번까지 N명의 사람이 원을 이루면서 앉아있고, 양의 정수 K(≤ N)가 주어진다. 이제 순서대로 K번째 사람을 제거한다. 한 사람이 제거되면 남은 사람들로 이루어진 원을 따라 이 과정을 계속해 나간다. 이 과정은 N명의 사람이 모두 제거될 때까지 계속된다. 원에서 사람들이 제거되는 순서를 (N, K)-요세푸스 순열이라고 한다. 예를 들어 (7, 3)-요세푸스 순열은 <3, 6, 2, 7, 5, 1, 4>이다.</p>

<p>N과 K가 주어지면 (N, K)-요세푸스 순열을 구하는 프로그램을 작성하시오.</p>

## Approach
이 문제는 문제의 요구사항을 정확히 구현하는 **구현 (Implementation)** 능력이 중요합니다.

특별한 알고리즘보다는 문제에서 주어진 조건을 빠짐없이 코드로 옮기는 과정이 필요하며, 예외 케이스 처리에 유의해야 합니다.

## Solution
```python
from collections import deque

N, K = map(int, input().split())

# Initialize a queue with numbers from 1 to N
queue = deque(range(1, N+1))
result = []

while queue:
    # Move the first K-1 elements to the back of the queue
    for _ in range(K-1):
        queue.append(queue.popleft())
    
    # Remove the K-th person and add to the result
    result.append(queue.popleft())

# Print the result in the required format
print('<' + ', '.join(map(str, result)) + '>')

```
