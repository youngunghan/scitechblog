## Introduction
백준 온라인 저지(BOJ)의 **[요세푸스 문제 0](https://www.acmicpc.net/problem/11866)** 문제 풀이입니다.

## Problem Description

<p>요세푸스 문제는 다음과 같다.</p>

<p>1번부터 N번까지 N명의 사람이 원을 이루면서 앉아있고, 양의 정수 K(≤ N)가 주어진다. 이제 순서대로 K번째 사람을 제거한다. 한 사람이 제거되면 남은 사람들로 이루어진 원을 따라 이 과정을 계속해 나간다. 이 과정은 N명의 사람이 모두 제거될 때까지 계속된다. 원에서 사람들이 제거되는 순서를 (N, K)-요세푸스 순열이라고 한다. 예를 들어 (7, 3)-요세푸스 순열은 <3, 6, 2, 7, 5, 1, 4>이다.</p>

<p>N과 K가 주어지면 (N, K)-요세푸스 순열을 구하는 프로그램을 작성하시오.</p>

## Approach
### 1. 문제 분석 (Problem Analysis)
이 문제는 원형으로 앉은 사람들을 순서대로 제거하는 **요세푸스 순열(Josephus Permutation)**을 구하는 문제입니다.
- $N$명의 사람이 원을 이루고 있습니다.
- $K$번째 사람을 계속해서 제거합니다.
- 원형 구조를 시뮬레이션하기 위해 **큐(Queue)** 자료구조가 적합합니다.
- 큐의 앞(Front)에서 사람을 꺼내어 뒤(Back)로 보내는 과정을 반복하면 원형 회전을 구현할 수 있습니다.

### 2. 해결 아이디어 (Solution Idea)
1.  **큐 초기화**: 1부터 $N$까지의 숫자를 큐에 넣습니다.
2.  **회전 (Rotation)**: $K-1$번 동안 큐의 맨 앞 사람을 꺼내어 맨 뒤로 보냅니다.
3.  **제거 (Removal)**: $K$번째 사람(현재 큐의 맨 앞)을 꺼내어 결과 리스트에 추가합니다.
4.  **반복**: 큐가 빌 때까지 위 과정을 반복합니다.

### 3. 코드 분석 (Code Analysis)
- Python의 `collections.deque`를 사용하여 큐를 구현합니다. `deque`는 양쪽 끝에서의 삽입/삭제가 $O(1)$로 매우 효율적입니다.
- `queue.popleft()`로 사람을 꺼내고, `queue.append()`로 뒤로 보냅니다.
- 마지막에 출력 형식을 `<...>` 형태로 맞춰줍니다.

## Solution
```python
import sys
from collections import deque

# 입력 받기
input = sys.stdin.readline
N, K = map(int, input().split())

# 1부터 N까지의 사람을 큐에 초기화
queue = deque(range(1, N + 1))
result = []

while queue:
    # K-1번 회전: 맨 앞 사람을 맨 뒤로 이동
    for _ in range(K - 1):
        queue.append(queue.popleft())
    
    # K번째 사람 제거 및 결과 저장
    result.append(queue.popleft())

# 결과 출력 형식 맞추기 (<3, 6, 2, ...>)
print('<' + ', '.join(map(str, result)) + '>')
```

## Complexity Analysis
- **Time Complexity**: $O(N \times K)$
    - 총 $N$명의 사람을 제거해야 합니다.
    - 한 명을 제거할 때마다 $K-1$번의 회전 연산이 발생합니다.
    - $N, K \le 1,000$이므로 최대 연산 횟수는 약 $10^6$번으로, 시간 제한(2초) 내에 충분합니다.
- **Space Complexity**: $O(N)$
    - 큐에 $N$명의 사람을 저장해야 합니다.

## Conclusion
큐(Queue)의 선입선출(FIFO) 성질과 회전(Rotation)을 이용하여 원형 문제를 간단하게 시뮬레이션할 수 있음을 보여주는 기초적인 문제입니다. `deque`의 활용법을 익히기에 좋습니다.
